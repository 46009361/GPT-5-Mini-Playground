# GPT-5 Playground

Bypass GPT-5 resource limits imposed on chatgpt.com! Inspired by [OpenAI Playground](https://platform.openai.com/playground).

This is a large language model that "thinks" for a while, so please expect slower responses.

I thank [Flexbox Froggy](https://flexboxfroggy.com/) for teaching me how to use CSS flex.

## Input fields

* System prompt: The instructions given for the model to follow.
   * Example: You are a helpful assistant.
* Your message: The query you decide to use.
   * Example: Is "dog" or "cat" the subject of the sentence, "The dog walked in the park."?
   * Anything inappropriate for teenagers may trigger extra filters on top of the model's moderation layer, which I can't control
* Response from GPT-5: The model's output.
